# Copycasts

Crawling railcast video and download to your local

## Installation

Add this line to your application's Gemfile:

    gem 'copycasts'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install copycasts

## Usage

Run using this command in your terminal:

    copycasts-start-crawl
